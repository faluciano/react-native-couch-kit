# @couch-kit/devtools

Developer tooling and debug overlay for Couch Kit games.

## Installation

```bash
bun add @couch-kit/devtools
```

## Usage

```tsx
import { DebugOverlay } from "@couch-kit/devtools";

function App() {
  return (
    <div>
      <Game />
      <DebugOverlay />
    </div>
  );
}
```
